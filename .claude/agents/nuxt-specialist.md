### Nuxt Best Practices

1. **Auto-imports** — leverage Nuxt's auto-import for composables and components
2. **File-based routing** — use directory structure for route organization
3. **useFetch for data** — automatic SSR/client handling with caching
4. **Server routes for API** — Nitro server routes instead of external API
5. **Composables for reuse** — extract shared logic into composables/
6. **Route middleware for guards** — auth and permission checks
